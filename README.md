💙 SafeSpace AI

SafeSpace AI is an AI-powered emotional wellness platform designed to provide a private and supportive digital environment for users to reflect on emotions, track moods, journal thoughts, and receive AI-powered guidance.

Features

- 🤖 AI Companion
- 😊 Mood Tracker
- 📔 Private Journal
- 💌 Letter Never Sent
- 🌿 Relaxation Corner
- ✨ Daily Motivation
- 📊 Mood Insights Dashboard

Technologies Used

- Python
- Streamlit
- Google Gemini API
- SQLite
- Pandas
- Plotly

Installation

pip install streamlit
pip install google-generativeai
pip install pandas
pip install plotly

Run Project

streamlit run app.py

Project Objective

To create a safe digital space where users can express thoughts, reflect on emotions, monitor mood patterns, and receive supportive AI-powered assistance.
